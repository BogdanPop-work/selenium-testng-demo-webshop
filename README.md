# Selenium TestNG Demo Web Shop

UI automation framework built with:

- Java
- Selenium WebDriver
- TestNG
- Maven
- WebDriverManager

## Application Under Test

https://demowebshop.tricentis.com

## Project Goals

- Build a scalable Selenium automation framework
- Implement Page Object Model (POM)
- Support cross-browser execution
- Add reporting and screenshots
- Integrate with GitHub Actions CI/CD
